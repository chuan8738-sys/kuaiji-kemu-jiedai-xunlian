# 会计科目借贷方向互动训练系统 V1.0

这是一款面向会计基础学习者的互动练习软件，主要训练科目类别判断、增减方向判断，以及借方、贷方的选择。

V1.0 提供一题三判、方向选择、即时反馈、错题复盘、简单统计和练习结束反馈等功能。

## 下载

请前往 [Releases 最新版本页面](../../releases/latest)，下载：

- `kuaiji-kemu-jiedai-xunlian-V1.0.zip`（学生可运行版）
- `kuaiji-kemu-jiedai-xunlian-V1.0-SHA256.txt`（完整性校验）

请不要下载 GitHub 自动生成的 `Source code` 压缩包，它不是学生可运行版。

## 使用方法

1. 将下载的 ZIP 完整解压，不要直接在压缩包中运行。
2. 打开解压后的文件夹。
3. 双击带绿色播放图标的 `开始游戏.exe`。
4. 启动器会自动打开默认浏览器并进入练习。

学生电脑不需要安装 Node.js、Python 或数据库，也不需要管理员权限。

如需从桌面进入，请先把解压后的整个文件夹放到固定位置，再双击 `创建桌面快捷方式.cmd`。创建快捷方式后，请不要移动或删除原文件夹。

## 安全提示

`开始游戏.exe` 目前没有商业代码签名证书，部分学校电脑可能显示“未知发布者”或被安全策略拦截。

遇到拦截时，请由教师或管理员先核对发布页提供的 SHA256 校验值，再使用包内的 `StartGame.cmd` 备用入口。请不要自行关闭或绕过学校电脑的安全策略。

## 数据说明

启动器只在本机 `127.0.0.1` 提供练习页面。练习记录、错题和反馈保存在当前浏览器的本地存储中，不会自动汇总到教师电脑或上传到本仓库。
